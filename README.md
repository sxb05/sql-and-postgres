## Overview

This repository documents my **daily learning and hands-on practice** in **Python backend development** using **FastAPI** and **PostgreSQL**.

Each commit represents **end-of-day progress**, including newly learned concepts, practical implementations, experiments, and refactoring as my understanding improves.  
The purpose of this repository is to **track real learning progress** and build a strong backend foundation.

---

## Technology Stack

- **Programming Language:** Python  
- **Backend Framework:** FastAPI  
- **Database:** PostgreSQL  
- **Data Validation:** Pydantic  
- **API Testing:** Postman  
- **Database Management:** pgAdmin  
- **Version Control:** Git & GitHub  
- **Editor:** Visual Studio Code

### Environment Setup
- Python installation on Windows and macOS
- Visual Studio Code setup for backend development
- Python virtual environments
- Dependency management using `pip`

### FastAPI Development
- Creating and running FastAPI applications
- Path operations (GET, POST, PUT, DELETE)
- Understanding path operation order and routing behavior
- Handling HTTP requests and responses
- API testing using Postman
- Request and response schema validation with Pydantic

- Working with in-memory data structures
- Retrieving single and multiple resources
- Updating and deleting resources
- Custom HTTP status codes
- Automatic API documentation using Swagger UI and ReDoc
- Python package structure and imports
